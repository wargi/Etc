# a to b #

## 1. 문제
- 변수 a, b를 만들고, 각 각에 숫자를 입력받아 주세요.
- 그 후, 재귀 호출을 이용하여 a -> b -> a까지 출력해주세요.

## 2. 입력
- 변수 a, b를 만들고, 각 각에 숫자를 입력받아 주세요.

## 3. 출력
- 재귀 호출을 이용하여 a -> b -> a까지 출력해주세요.

## 4. 예제 입력
```
1 5
```

## 5. 예제 출력
```
1 2 3 4 5 4 3 2 1
```

## 6. 예제 입력

```
5 9
```

## 7. 예제 출력

```
5 6 7 8 9 8 7 6 5
```

## 8. 예제 입력

```
2 7
```

## 9. 예제 출력

```
2 3 4 5 6 7 6 5 4 3 2
```

## 10. 코드

```c++
#include <iostream>
using namespace std;

int aToB(int a, int b) {
    cout << a << " ";
    if (a == b) return 0;
    aToB(a + 1, b);
    cout << a << " ";
}

int main()
{
    int a, b;
    cin >> a >> b;

    aToB(a, b);
}
```
