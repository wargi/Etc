# DAT 익혀보기 #

## 1. 문제
- 숫자 8개를 입력받습니다 (0 <= n <= 9)

```
1 9 9 3 1 1 0 2

1. 입력받은 숫자를 DAT배열에 추가합니다.
0: 1
1: 3
2: 1
3: 1
4: 0
5: 0
6: 0
7: 0
8: 0
9: 2

2. 결과를 순서대로 DAT 배열 내부 값이 index 1의 값이 1이면 1회출력,
   index 2의 값이 2이면 2회 출력 아래와 같이 출력해줍니다.
   이렇게 하면 위의 예제는 아래의 결과가 나옵니다.
   0 1 1 1 2 3 9 9
```

## 2. 입력
- 숫자 8개를 입력받습니다 (0 <= n <= 9)

## 3. 출력
- 입력받은 숫자를 DAT배열에 추가하여 최종결과를 출력해주세요.

## 4. 예제 입력
```
1 9 3 1 0 1 0 7
```

## 5. 예제 출력
```
0 0 1 1 1 3 7 9
```

## 6. 코드
```c++
#include <iostream>
using namespace std;

int main()
{
    int bucket[10] = { 0 };
    int vect[8];
    for (int i = 0; i < 8; i++) {
        cin >> vect[i];
        bucket[vect[i]]++;
    }
    
    for (int i = 0; i < 10; i++) {
        for (int j = 0; j < bucket[i]; j++) {
            cout << i << " ";
        }
    }
}
```
