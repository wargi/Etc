# 마스킹 된 부분의 합 구하기

## 1. 문제
```
3 x 3 사이즈의 2차원 배열을 아래와 같이 하드코딩해주세요.
1 7 4
2 8 5
3 9 6

문제 설명)
그리고, 마스킹된 배열(3x3)을 입력 받아주세요.
1 0 1
0 1 0
1 0 1

마스킹 된 부분(배열의 값이 1인 부분)에 해당하는 값들 만 더하여 결과를 출력해주세요.
1 7 4    1 0 1    1 0 4
2 8 5 -> 0 1 0 -> 0 8 0 -> 1 + 4 + 8 + 3 + 6 -> 22를 출력
3 9 6    1 0 1    3 0 6
```

## 2. 입력
- 마스킹된 배열(3x3)을 입력 받아주세요.

## 3. 출력
- 마스킹 된 부분(배열의 값이 1인 부분)에 해당하는 값들 만 더하여 결과를 출력해주세요.

## 4. 예제 입력
```
1 0 1
0 1 0
1 0 1
```

## 5. 예제 출력
```
22
```

## 6. 코드
```c++
#include <iostream>
using namespace std;

int main()
{
    int check[3][3], sum = 0;
    int vect[3][3] = {
    1,7,4,
    2,8,5,
    3,9,6
    };
    
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
           cin >> check[i][j];
        }
    }

    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            if (check[i][j] == 1) sum += vect[i][j];
        }
    }
    cout << sum;
}
```
