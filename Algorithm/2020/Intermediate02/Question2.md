# 7로 바꾸어 출력하기

## 1. 문제
```
아래의 배열을 하드코딩 해주세요.
3 5 4 2 6 6 5
```

- 마스킹 배열을 입력받고, 마스킹된 부분은 7로 아닌 부분은 0으로 출력하기.

## 2. 입력
- 1차원 배열(1x7)을 입력 받는다.

## 3. 출력
- 마스킹된 부분을 7로 바꾸어 출력한다.

## 4. 예제 입력
```
0 1 1 0 1 1 0
```

## 5. 예제 출력
```
0770770
```

## 6. 코드
```c++
#include <iostream>
using namespace std;

int vect[7] = { 3, 5, 4, 2, 6, 6, 5 };

int main()
{
    int bit[7];
    

    for (int i = 0; i < 7; i++) {
        cin >> bit[i];
    }

    for (int i = 0; i < 7; i++) {
        if (bit[i] == 1) cout << "7";
        else cout << "0";
    }
}
```
