# heap 익혀보기 #

## 1. 문제

- int a, b를 가진 구조체를 heap을 이용하여 만들어주세요.
- 그리고 숫자 2개를 a, b에 입력 받고 5를 더한 결과 값을 출력해주세요.

## 2. 입력
- 숫자 2개를 a, b에 입력 받는다.

## 3. 출력
- 5를 더한 결과 값을 출력해주세요.

## 4. 예제 입력
```
1 5
```

## 5. 예제 출력
```
2 10
```

## 6. 예제 입력

```
4 8
```

## 7. 예제 출력

```
9 13
```

## 8. 코드

```c++
#include <iostream>
using namespace std;

struct BBQ {
    int a, b;
};

int main()
{
    BBQ* x = new BBQ;
    
    cin >> x->a >> x->b;
    cout << x->a + 5 << " " << x->b + 5;
}
```
