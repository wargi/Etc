# Calculator

## 1. 문제

- 덧셈, 뺄셈, 곱셈, 나눗셈을 할 수 있는 계산기 프로그램을 만드세요.

## 2. 입력
- 첫째 줄: 숫자 a가 입력
- 둘째 줄: +, -, /, * 연산자가 입력된다.
- 마지막 줄: 숫자 b가 입력된다.

## 3. 출력

- a와 b를 주어진 연산자를 활용하여 연산한 결과를 출력해라.


## 4. 예제 입력
```
5
+
5
```

## 5. 예제출력

```
10
```

## 6. 예제 입력

```
5
-
12
```

## 7. 예제출력

```
-7
```

## 8. 예제 입력

```
2
*
105
```

## 9. 예제출력

```
210
```

## 10. 예제 입력

```
125
/
5
```

## 11. 예제출력

```
25
```

## 12. 코드

```c++
#include <stdio.h>

int main() {
  int a, b;
  char cal;
  scanf("%d %c %d", &a, &cal, &b);
  
  if(cal == '+') a += b;
  if(cal == '-') a -= b;
  if(cal == '*') a *= b;
  if(cal == '/') a /= b;
  
  printf("%d", a);

  return 0;
}
```
