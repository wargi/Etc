# P Factorization

## 1. 문제

- 정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.
- 소인수란 소수인 인수(약수)를 의미한다.  

## 2. 입력
- 첫째 줄: 정수 N (1 ≤ N ≤ 10,000,000)

## 3. 출력

- N의 소인수를 한 줄에 하나씩 오름차순으로 출력한다..


## 4. 예제 입력
```
72
```

## 5. 예제 출력
```
2
2
2
3
3
```

## 6. 예제 입력

```
3
```

## 7. 예제 출력

```
3
```

## 8. 예제 입력

```
6
```

## 9. 예제 출력

```
2
3
```

## 10. 예제 입력

```
9991
```

## 11. 예제 출력

```
97
103
```

## 8. 코드

```c++
#include <stdio.h>

int main() {
  int num;
  scanf("%d", &num);
  for(int i = 2; i <= num; i++) {
    if (num % i == 0) {
      printf("%d\n",i);
      num /= i;
      i = 1;
    }
  }
  return 0;
}
```
